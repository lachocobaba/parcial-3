# Gestión de Calificaciones

## Descripción
Este programa en C++ permite registrar las calificaciones de estudiantes utilizando arreglos. Calcula promedios, identifica la nota más alta y más baja, y muestra cuántos estudiantes aprobaron o reprobaron.

## Ejemplo de ejecución

Cantidad de estudiantes: 2

Estudiante 1:
80 90 70 60

Estudiante 2:
50 40 60 70

Resultados:
Promedio general: 65
Nota mayor: 90
Nota menor: 40
Aprobados: 1
Reprobados: 1

 Explicación
Se utilizan arreglos bidimensionales para almacenar las notas de cada estudiante. Luego, se recorren con ciclos para calcular promedios y estadísticas.
